# AgroCast: Rice Yield & Climate Prediction

A machine learning project that predicts rice crop yields using historical agricultural and climate data. The platform combines data analysis, predictive modeling, and an interactive web dashboard to evaluate the impact of environmental factors on agricultural productivity.

## Live Demo
https://crop-yield-forecasting-assam.vercel.app/

## Key Features
- Rice yield prediction using Multiple Linear Regression
- Climate-based scenario analysis (heatwave, drought, and monsoon simulations)
- Interactive web dashboard with real-time calculations
- Data analysis and model development using Python
- Historical agricultural and climate data integration

## Tech Stack
- Python (Pandas, NumPy, Scikit-learn)
- HTML, CSS, JavaScript
- Chart.js
- Jupyter Notebook
