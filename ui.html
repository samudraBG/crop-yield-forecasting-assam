About This Model

This prediction tool uses an XGBoost machine learning model trained on historical rice yield data with the following environmental parameters:

  • Maximum Temperature (T2M_MAX): The highest temperature recorded during the growing season
  • Minimum Temperature (T2M_MIN): The lowest temperature recorded during the growing season
  • Relative Humidity (RH2M): The average relative humidity during the growing season

For best results, input values that represent average conditions over the entire growing season.
